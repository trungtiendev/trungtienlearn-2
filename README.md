# TrungTienLearn - Educational Platform

## Công nghệ
- **Next.js 14+** (App Router, TypeScript)
- **Supabase** (Database, Auth, Storage)
- **TailwindCSS** + **Shadcn/ui**
- **Stripe** / **VNPay** (Thanh toán)
- **GitHub** (Version Control)

## Domain
- `trungtienlearn.com`

## Tính năng
1. **Authentication**: Đăng nhập/Đăng ký (Email, Google OAuth)
2. **Blog**: Đăng bài viết với Rich Text Editor
3. **E-commerce**: Bán sản phẩm số (ebook, template, course access)
4. **LMS**: Khóa học online với tiến độ theo dõi
5. **Payment**: Tích hợp Stripe & VNPay
